# Muxi\_k's NeoVim configuration


## 前言

轻量级的nvim配置，仅有少量的偏好设置。




## 依赖 

- nodejs 
- python 
- pynvim(pip)
- npm
- [Nerd Font](https://www.nerdfonts.com/)
- fzf (文件查找)
- ag  (搜索)

## 安装


```bash
$ git clone https://github.com/muxik/nvim.git
```

```
$ cp nvim ~/.config 
```

A: 修改 plug/plug-list.vim 对应的插件
